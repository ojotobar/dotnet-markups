Tiered Rate Copy Tool

This tool copies existing tiered rate configurations from a source payer to a target payer.

Only tasks that already have tiered rate setups on the source payer are considered.

To prevent overwriting configurations, any task that already has a tiered rate setup on the target payer will be skipped automatically.

How it works

  1. The tool scans the source payer for tasks with configured tiered rates.
  2. For each task found:
    • If the target payer does not have tiered rates for that task, the configuration is copied.
    • If the target payer already has tiered rates for that task, the task is skipped.
  3. No existing tiered rate configurations are modified or overwritten.

Important

  • This tool does not overwrite existing tiered rate setups on the target payer.
  • Tasks without tiered rate configurations on the source payer are not copied.
